# IP_AddressScanner
This program will expect to proviode the list of IP addresses as YAML input
Eample YAML file:

ips:
  - 202.122.21.106
  - 202.122.21.107
  - 202.122.21.108
  - 2002.122.21.10

  Run the program:
   python IPScanner.py

Prerequisite: 

install the package using pip for yaml requests etc.
provide the list of IPs to scan in YAML file.
